WordPress doesn't allow the .svg,.webp format for media upload and .ttf,.otf fonts files upload gives an error. Active the Plugin and Enjoy. uploads your Svg, Webp Images, TTF, OTF fonts file Safely without Erros.

== Description ==

Enhance your WordPress media capabilities with "Active the Plugin and Enjoy." This plugin extends your media library to support not only SVG and WebP images but also TTF, OTF, EOT, and WOFF font files. Safety is our top priority; that's why we've included an SVG sanitization feature to keep your site secure while you enjoy broader media upload options. Take control of your media and start uploading without errors today!

== Installation ==

1. Login in your WordPress admin panel.   
2. Navigate to Plugins => Add New from left menu bar.
3. Now search "CITS Support svg, webp Media Upload" in search box.
4. Click on Install Now button
5. Then activate the plugin.
6. Now you are ready to upload svg, webp images and ttf,otf etc fonts files.
